# 🕹️ Laboratorio 2: Juego de Ping Pong con ESP32 y Matriz LED 8x8

Este laboratorio consiste en la implementación del clásico juego **Ping Pong** (segun mi criterio) utilizando un microcontrolador **ESP32** y una **Matriz LED de 8x8 bicolor**. El sistema integra control de hardware (multiplexación y transistores), lógica de estados de juego y manejo de periféricos mediante interrupciones y temporizadores.

En este repositorio se encuentra el codigo que hace funcionar al juego.
